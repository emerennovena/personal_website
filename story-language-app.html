Story-Based Language Learning App

Designed to immerse users in engaging stories that naturally teach language skills, combining narrative with effective learning techniques.

Project Overview

This app helps learners improve their language skills through interactive stories. By focusing on context and storytelling, users build vocabulary and grammar naturally.

Key Features

  • Immersive storytelling to enhance language retention
  • Integrated audio and text for better pronunciation

Project Details

Role: UX Designer & Developer
Timeline: 2 months
Tools Used: Figma, InkleWriter
